Função afim¶

Uma função afim, ou função do primeiro grau, é uma função do tipo:

f(x) = ax + b¶

Ou seja, uma função na qual a variável x está elevada à primeira potência.

O gráfico característico de uma função afim crescente é da forma:

In [2]:
x = linspace(-2,2,2)
plot(x,3*x+4)
title('Função Afim crescente')
Out[2]:
Text(0.5,1,'Função Afim crescente')

Por outro lado, o gráfico de uma função afim decrescente é da forma:

In [3]:
x = linspace(-2,2,2)
plot(x,-3*x+4)
title('Função Afim decrescente')
Out[3]:
Text(0.5,1,'Função Afim decrescente')

Função quadrática¶

Uma função quadrática, ou função do segundo grau, é uma função do tipo:

f(x) = ax² + bx + c¶

Ou seja, uma função na qual a variável x está elevada à segunda potência.

O gráfico característico de uma função quadrática com a > 0 é da forma:

In [4]:
x = linspace(-10,15,200)
plot(x,x**2-5*x+4, color="red")
title('Função Quadrática com a > 0')
Out[4]:
Text(0.5,1,'Função Quadrática com a > 0')

Por outro lado, o gráfico de uma função quadrática com a < 0 é da forma:

In [5]:
x = linspace(-15,10,200)
plot(x,-x**2-5*x+4, color="red")
title('Função Quadrática com a < 0')
Out[5]:
Text(0.5,1,'Função Quadrática com a < 0')

Função cúbica¶

Uma função cúbica, ou função do terceiro grau, é uma função do tipo:

f(x) = ax³ + bx² + cx + d¶

Ou seja, uma função na qual a variável x está elevada à terceira potência.

O gráfico característico de uma função cúbica é da forma:

In [6]:
x = linspace(-10,10,200)
plot(x,x**3, color="black")
title('Função Cúbica')
Out[6]:
Text(0.5,1,'Função Cúbica')

Função exponencial¶

Uma função exponencial é uma função do tipo:

f(x) = a x ¶

Ou seja, uma função na qual a variável x está localizada na potência e na base está um número real.

O gráfico característico de uma função exponencial com x > 0 é da forma:

In [7]:
x = linspace(-2,10,200)
plot(x,e**x, color="green")
title('Função Exponencial com x > 0')
Out[7]:
Text(0.5,1,'Função Exponencial com x > 0')

Por outro lado, o gráfico de uma função exponencial com x < 0 é da forma:

In [8]:
x = linspace(-2,10,200)
plot(x,e**-x, color="green")
title('Função Exponencial com x < 0')
Out[8]:
Text(0.5,1,'Função Exponencial com x < 0')

Função logarítmica¶

O logaritmo de um número é definido como o expoente ao qual se deve elevar a base a para obter o número x. Uma função logarítmica é uma função do tipo:

f(x) = log ax¶

O gráfico característico de uma função logarítmica positiva é da forma:

In [9]:
x = linspace(0.5,1000,200)
plot(x,log (x), color="orange")
title('Função Logarítmica positiva')
Out[9]:
Text(0.5,1,'Função Logarítmica positiva')

Por outro lado, o gráfico de uma função logarítmica negativa é da forma:

In [10]:
x = linspace(0.5,1000,200)
plot(x,-log (x), color="orange")
title('Função Logarítmica negativa')
Out[10]:
Text(0.5,1,'Função Logarítmica negativa')
